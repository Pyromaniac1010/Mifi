# MiFi (MVP)

A mobile-first financial freedom tracker:
- Transactions (income/expense, active/passive)
- Debt management (avalanche ranking, freedom date)
- Solvency score
- Mi assistant (4 personalities)
- localStorage persistence

## Run locally
```bash
npm install
npm run dev
```

## Deploy (GitHub → Vercel)
1. Push this repo to GitHub
2. On Vercel: Add New → Project → Import repo
3. Framework: Next.js (auto)
4. Deploy
```
